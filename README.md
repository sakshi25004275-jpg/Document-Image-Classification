# Document Image Classification using CNN

## Project Overview
This project focuses on classifying different types of documents using Deep Learning. A Convolutional Neural Network (CNN) model is trained on the RVL-CDIP dataset to identify document categories automatically.

## Problem Statement
To develop an AI-based system that can classify document images into different categories such as invoice, letter, form, resume, advertisement, etc.

## Dataset
Dataset Used: RVL-CDIP Small Dataset

Number of Classes: 16

Classes include:
- Invoice
- Letter
- Form
- Resume
- Advertisement
- Memo
- Scientific Report
- Presentation
- Email
- Handwritten
- Budget
- Questionnaire
- File Folder
- Specification
- News Article
- Scientific Publication

## Methodology
1. Dataset Collection
2. Image Preprocessing
3. Data Augmentation
4. CNN Model Training
5. Model Evaluation
6. Document Prediction

## Model Architecture
Convolutional Neural Network (CNN)

## Results
Training Accuracy: 83.50%

Validation Accuracy: 46.09%

## Prediction Example
Input Document: Fee Receipt PDF

Predicted Class: Invoice
